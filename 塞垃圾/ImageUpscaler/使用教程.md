# 图片放大处理工具 - 使用教程

## 一、软件简介

**图片放大处理工具 v1.0** 是一款功能强大的图片放大处理软件，支持多种常见图片格式，提供多种可自定义的放大方式。

### 主要功能
- 支持批量处理单张或多张图片
- 提供5种放大方式：修改尺寸、插值放大、AI智能放大、添加边框、组合处理
- 支持多种插值算法：最近邻、双线性、双三次、Lanczos
- 支持设置输出格式、质量和保存路径
- 提供命令行和图形界面两种操作模式
- 实时进度显示和详细处理日志

### 支持格式
JPG、JPEG、PNG、GIF、BMP、WEBP、TIFF

## 二、安装与运行

### 环境要求
- Windows 7/10/11
- Python 3.7 或更高版本（如使用源码运行）

### 方法一：直接运行EXE（推荐）
1. 解压软件包到任意文件夹
2. 双击运行 `ImageUpscaler.exe`（图形界面）或 `ImageUpscaler_CLI.exe`（命令行）

```bash
# 运行图形界面
python image_upscaler_gui.py

# 运行命令行版本
python image_upscaler.py
```

## 三、图形界面使用教程

### 1. 启动软件
双击 `ImageUpscaler.exe` 启动图形界面。

### 2. 添加图片
- 点击 **"添加文件"** 按钮选择单张或多张图片
- 或点击 **"添加文件夹"** 按钮批量导入整个文件夹
- 支持递归扫描子文件夹

### 3. 选择放大方式
在左侧选择一种放大方式：

| 方式 | 说明 | 适用场景 |
|------|------|----------|
| **修改尺寸** | 直接调整像素大小 | 快速调整尺寸 |
| **插值放大** | 使用插值算法放大 | 通用放大，质量较好 |
| **AI智能放大** | 深度学习超分辨率 | 需要高质量放大 |
| **添加边框** | 保持原图，添加边框 | 需要留白/边框效果 |
| **组合所有** | 依次应用所有方式 | 特殊需求 |

### 4. 设置参数

#### 通用参数
- **放大倍数**：设置放大比例（如 2.0 表示放大2倍）
- **输出格式**：png / jpg / webp / bmp
- **输出质量**：1-100，数值越高质量越好（仅对jpg/webp有效）
- **输出目录**：设置保存位置

#### 插值算法（插值放大模式）
- **最近邻(nearest)**：速度快，质量一般，有锯齿
- **双线性(bilinear)**：平衡速度和质量
- **双三次(bicubic)**：质量较好
- **Lanczos**：质量最好，速度稍慢（推荐）

#### 边框参数（添加边框模式）
- **边框大小**：设置边框宽度（像素）

#### 高级选项
- **锐化**：处理后应用锐化滤镜
- **降噪**：处理后应用降噪滤镜

### 5. 开始处理
点击 **"开始处理"** 按钮，右侧会显示实时进度和日志。

### 6. 查看结果
- 处理完成后，结果保存在设置的输出目录中
- 点击 **"查看报告"** 可查看详细处理报告

## 四、命令行使用教程

### 基本语法
```bash
python image_upscaler.py -i <输入路径> [选项]
```

### 常用命令示例

#### 1. 交互模式
```bash
python image_upscaler.py
```
不带参数启动，按提示操作。

#### 2. 放大单张图片
```bash
# 放大2倍（默认使用Lanczos插值）
python image_upscaler.py -i photo.jpg -s 2.0

# 放大3倍，使用双三次插值
python image_upscaler.py -i photo.jpg -m interpolate -s 3.0 --interp bicubic

# 放大2倍并锐化
python image_upscaler.py -i photo.jpg -s 2.0 --sharpen
```

#### 3. 批量处理文件夹
```bash
# 处理整个文件夹
python image_upscaler.py -i ./photos/ -m interpolate -s 2.0

# 递归处理子文件夹
python image_upscaler.py -i ./photos/ -m interpolate -s 2.0 -r
```

#### 4. 指定目标尺寸
```bash
# 设置目标宽度和高度
python image_upscaler.py -i photo.jpg -m resize --width 1920 --height 1080
```

#### 5. 添加边框
```bash
# 添加100像素的白色边框
python image_upscaler.py -i photo.jpg -m border -b 100 -f jpg
```

#### 6. 使用AI放大
```bash
# 使用AI模型放大2倍
python image_upscaler.py -i photo.jpg -m ai -s 2.0 --ai-model edsr
```

#### 7. 组合处理
```bash
# 使用所有方式组合处理
python image_upscaler.py -i photo.jpg -m all -s 2.0 --sharpen --denoise
```

### 命令行参数说明

| 参数 | 说明 | 示例 |
|------|------|------|
| `-i, --input` | 输入文件或文件夹路径（必填） | `-i photo.jpg` |
| `-m, --method` | 放大方式 | `-m interpolate` |
| `-s, --scale` | 放大倍数 | `-s 2.0` |
| `--width` | 目标宽度 | `--width 1920` |
| `--height` | 目标高度 | `--height 1080` |
| `--interp` | 插值算法 | `--interp lanczos` |
| `-b, --border` | 边框大小 | `-b 50` |
| `-f, --format` | 输出格式 | `-f png` |
| `-q, --quality` | 输出质量 | `-q 95` |
| `-o, --output` | 输出目录 | `-o output/` |
| `-r, --recursive` | 递归扫描子文件夹 | `-r` |
| `--sharpen` | 应用锐化 | `--sharpen` |
| `--denoise` | 应用降噪 | `--denoise` |
| `--ai-model` | AI模型 | `--ai-model edsr` |
| `--ai-scale` | AI放大倍数 | `--ai-scale 2` |

## 六、配置文件

软件会自动生成 `upscaler_config.json` 配置文件，保存用户的默认设置。

可以手动编辑此文件修改默认参数：
```json
{
  "scale_factor": 2.0,
  "interpolation": "lanczos",
  "border_size": 50,
  "output_format": "png",
  "quality": 95,
  "output_dir": "output_upscaled",
  "sharpen": false,
  "denoise": false
}
```

## 八、常见问题

### Q1: 处理后的图片质量不理想？
- 尝试使用 `Lanczos` 插值算法
- 开启 `--sharpen` 锐化选项
- 使用AI放大模式（需要下载模型文件）
- 提高输出质量参数 `-q 100`

### Q2: AI放大功能无法使用？
AI功能需要下载预训练模型文件：
1. 创建 `models/` 目录
2. 下载模型文件（如 `EDSR_x2.pb`）
3. 放入 `models/` 目录

模型下载地址：
- OpenCV官方模型仓库

### Q3: 批量处理时某些文件失败？
- 检查文件是否为支持的格式
- 检查文件是否损坏
- 查看日志文件获取详细错误信息

### Q4: 如何保持原始宽高比？
软件默认会保持宽高比，只需设置放大倍数即可。

## 十、技术说明

### 插值算法对比

| 算法 | 速度 | 质量 | 特点 |
|------|------|------|------|
| 最近邻 | 最快 | 一般 | 边缘锯齿明显 |
| 双线性 | 快 | 较好 | 平滑，可能模糊 |
| 双三次 | 中等 | 好 | 平衡选择 |
| Lanczos | 稍慢 | 最好 | 保留细节（推荐） |

### AI模型对比

| 模型 | 速度 | 质量 | 特点 |
|------|------|------|------|
| EDSR | 慢 | 最好 | 深度网络，细节丰富 |
| ESPCN | 快 | 较好 | 轻量级，适合实时 |
| FSRCNN | 较快 | 好 | 快速超分辨率 |
| LapSRN | 中等 | 好 | 渐进式上采样 |
